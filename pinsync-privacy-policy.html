Privacy Policy

Last updated: March 26, 2026  ·  Effective immediately

1. Overview

PinSync ("the App", "we", "us") is a private tool used exclusively by the store owner to automatically publish Shopify product listings as Pinterest pins. This policy explains what data is accessed, how it is used, and how it is protected.

2. Data We Access

PinSync accesses the following data from your Shopify store via the Admin API:

  • Product titles — used as the Pinterest pin title
  • Product images — used as the Pinterest pin image
  • Product prices — included in the pin description
  • Product handles / URLs — used as the destination link on the pin
⚠️ PinSync does not access customer data, order history, payment information, or any personal information about your customers.

3. Data We Do Not Collect

  • Customer names, emails, or contact details
  • Order data or transaction history
  • Financial or billing information
  • Any personally identifiable information (PII)
  • Inventory levels or supplier data

4. How Data Is Used

Product data fetched from Shopify is used solely for the purpose of creating Pinterest pins on the connected Pinterest Business account owned and operated by the same store owner. Data is:

  • Transmitted directly to the Pinterest API (v5) via HTTPS
  • Not stored on any third-party server
  • Not shared with any other parties
  • Not used for advertising, analytics, or profiling

5. Data Storage & Retention

PinSync operates as a client-side browser application. No product data is stored in a database or on a server. All data processing occurs in the user's browser session and is discarded when the session ends.

API credentials (Shopify token, Pinterest token) are entered by the store owner and are only used to make API calls during active sessions. They are not transmitted to or stored by any PinSync service.

6. Third-Party Services

PinSync connects to the following third-party APIs on behalf of the store owner:

  • Shopify Admin API — to read product data. Subject to Shopify's Privacy Policy.
  • Pinterest API v5 — to create pins. Subject to Pinterest's Privacy Policy.

7. Security

All API communications are made over HTTPS (TLS). API tokens entered into the app are only used in-memory during the active browser session. Store owners are responsible for keeping their API credentials secure and revoking them if compromised.

8. GDPR & CCPA

PinSync does not process any personal data of end-users, customers, or third parties. As such, GDPR and CCPA obligations regarding personal data do not apply to PinSync's operation. The app only processes publicly available product catalog data belonging to the store owner.

9. Children's Privacy

PinSync is a business tool intended solely for use by store owners and operators. It is not directed at and does not knowingly collect data from children under the age of 13.

10. Changes to This Policy

We may update this Privacy Policy from time to time. The "Last updated" date at the top of this page will reflect any changes. Continued use of the App after changes constitutes acceptance of the revised policy.

11. Contact

If you have any questions about this Privacy Policy, please contact the app owner at the email address registered with your Shopify account.
